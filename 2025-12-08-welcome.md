### [cite_start]1. Multi-Sensor Fusion & AI [cite: 15, 49]
Developing intelligent detection and classification systems for **Anti-Drone** applications.
* [cite_start]**Sensor Fusion:** Integrated **CW, FMCW Radar, and RF Receivers** to improve detection accuracy by 30% compared to single sensors[cite: 174].
* [cite_start]**AI Algorithm:** Proposed a **CNN-based classification model** using time-series overlaid FMCW Range-Doppler images[cite: 30, 31].
* [cite_start]**Data Construction:** Built a time-synchronized multi-sensor dataset for drone classification[cite: 36].

### [cite_start]2. UAM Communication & Network Analysis [cite: 23, 333]
Analyzing 3D communication coverage for Urban Air Mobility utilizing real-terrain data.
* [cite_start]**Performance Visualization:** Visualized **SINR, Outage Probability, and SNR** in complex urban environments using **MATLAB**[cite: 334].
* [cite_start]**Network Optimization:** Analyzed the effect of **CoMP (Coordinated Multi-Point)** and network redundancy (Dual-Link) to ensure high-reliability communication[cite: 350, 426].
* [cite_start]**Simulation:** Developed a GUI-based simulator for UTM (UAS Traffic Management) communication performance analysis[cite: 391].

### [cite_start]3. Radar System Design & Tracking [cite: 286]
Designing hardware and software architectures for real-time target tracking.
* [cite_start]**System Design:** Developed an automatic target tracking system using **1D FMCW MIMO Radar** and camera/motor synchronization[cite: 302, 330].
* [cite_start]**Signal Processing:** Implemented CFAR and MUSIC algorithms for precise target localization[cite: 327].
